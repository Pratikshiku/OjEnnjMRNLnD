### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 试卷
5. 论坛
6. 公告
7. 三国视频
8. 三国图文
9. 三国资讯
10. 个人中心
11. 三国视频收藏
12. 三国图文收藏
13. 考试记录
14. 错题表


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 基础数据管理（帖子类型管理、公告类型管理、科目管理、三国视频类型管理、三国图文类型管理、三国资讯类型管理）
5. 论坛管理
6. 公告管理
7. 三国视频管理
8. 三国视频留言管理
9. 三国视频收藏管理
10. 三国图文管理
11. 三国图文留言管理
12. 三国图文收藏管理
13. 三国资讯管理
14. 用户管理
15. 试卷管理
16. 试题管理
17. 考试记录
18. 错题本
19. 轮播图管理
